# RemoteShop

A modern E-Commerce REST API built with FastAPI, SQLAlchemy, and MySQL. The project demonstrates production-oriented backend development practices including authentication, authorization, relational database design, dependency injection, schema validation, and modular application architecture.

## Authentication Flow

1. User registers.
2. Password is hashed before storage.
3. User logs in.
4. JWT Access Token is generated.
5. Token is sent with protected requests.
6. Current user is resolved through dependency injection.
7. Role-based permissions are enforced on protected endpoints.

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd RemoteShop
```

Create virtual environment:

```bash
python -m venv venv
```

Activate environment:

In MacOS/Linux : 

```bash
source venv/bin/activate
```

In Windows : 

```bash
source venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create environment file:

```env
DATABASE_URL=mysql+pymysql://username:password@localhost/remoteshop

SECRET_KEY=your-secret-key

ALGORITHM=HS256

ACCESS_TOKEN_EXPIRE_MINUTES=30
```


Start server:

```bash
uvicorn app.main:app --reload
```
